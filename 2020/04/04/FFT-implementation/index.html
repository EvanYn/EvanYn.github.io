本文分析 FFT 的含义以及如何在 MATLAB 实现 FFT 运算。

DTFT 与 DFT

假设长度为 NNN 的离散信号为 x[n],(0≤n≤N−1).x[n], (0 \leq n \leq N-1).x[n],(0≤n≤N−1). 由 DTFT 的表达式：

X(f)=∑n=−∞∞x[n]e−j2πfn=∑n=0N−1x[n]e−j2πfn,f∈(−∞,∞)(1)X(f) = \sum_{n = -\infty }^{\infty }x[n]e^{-j2\pi f n} = \sum_{n = 0 }^{N-1 }x[n]e^{-j2\pi f n}, \quad f \in (-\infty ,\infty ) \tag{1} X(f)=n=−∞∑∞​x[n]e−j2πfn=n=0∑N−1​x[n]e−j2πfn,f∈(−∞,∞)(1)

由 KKK 点 DFT 的表达式：

X[k]=∑n=0N−1x[n]e−j2πk/Kn,k∈[0,K−1](2)X[k] = \sum_{n = 0 }^{N-1 }x[n]e^{-j2\pi k/K n}, \quad k \in [0, K-1] \tag{2} X[k]=n=0∑N−1​x[n]e−j2πk/Kn,k∈[0,K−1](2)

比较 (1)式和 (2)式可得以下关系：

X[k]=X(f)∣f=k/K,k∈[0,K−1](3)X[k] = X(f)|_{f = k/K}, \quad k \in [0, K-1] \tag{3} X[k]=X(f)∣f=k/K​,k∈[0,K−1](3)

即 KKK 点 DFT X[k]X[k]X[k] 为在 DTFT X(f)X(f)X(f) 上一个周期 f∈[0,1)f \in [0,1)f∈[0,1) 内的等间隔采样序列。由于 DTFT 是周期的，DFT 也是周期的，周期为 KKK。

通常在绘制信号的频谱图时，将零频移至中心。由于 DTFT 的周期性，DFT 在区间 f∈(0.5,1]f \in (0.5,1]f∈(0.5,1] 和 f∈(−0.5,0]f \in (-0.5,0]f∈(−0.5,0] 对应的值相同。因此 DFT 的上半部分（ k>⌈K/2⌉k > \left \lceil K/2 \right \rceilk>⌈K/2⌉ ）通常画在前边，另一半画在后边。

MATLAB 实现

一、 创建一单频信号 x[n]=ej2πf0n,(0≤n≤N−1)x[n] = e^{j2\pi f_0 n}, (0 \leq n \leq N-1)x[n]=ej2πf0​n,(0≤n≤N−1)

二、确定 DFT 点数 KKK 并计算出频率轴，注意让零频移至中心。

三、调用 fft 函数。因为 MATLAB 默认的频率设置基于公式(3)，为了让频谱与第二步设置的频率轴对应，我们需要用 fftshift 函数对 fft 输出结果排序。另外，为了让最大值固定为 1，将输出除于序列长度 NNN.

四、绘制频谱图。
