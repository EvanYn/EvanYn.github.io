为什么 FFT 能分析出信号的频谱？
2019-11-08|信号处理
字数总计: 2.3k|阅读时长: 10 分钟

此问题在信号与系统或数字信号处理中，已有详尽的数学解释，在此仅通过一个简化的例子，对 FFT 建立直观的认识。

问题描述

单频信号的 DTFT

式(1)中的 x[n]x[n]x[n]由多个单频信号叠加而成，首先对某一特定单频信号的 DTFT 做具体考查。由 DTFT 变换式

X(f)=∑n=−∞∞x[n]e−j2πfn,f∈(−∞,∞)(2)X(f) = \sum_{n = -\infty }^{\infty }x[n]e^{-j2\pi f n}, \quad f \in (-\infty ,\infty ) \tag{2} X(f)=n=−∞∑∞​x[n]e−j2πfn,f∈(−∞,∞)(2)

因为 X(f)X(f)X(f) 的周期为 1，所以一般取 X(f)X(f)X(f) 主周期区间为 −0.5⩽f⩽0.5-0.5 \leqslant f \leqslant 0.5−0.5⩽f⩽0.5.

X1(f)=Asin⁡[π(f−f0)N]sin⁡[π(f−f0)]e−jπ(f−f0)(N−1)(4)X_1(f) = A\frac{\sin[\pi (f-f_0)N]}{\sin[\pi (f-f_0)]}e^{-j\pi(f-f_0)(N-1)} \tag{4} X1​(f)=Asin[π(f−f0​)]sin[π(f−f0​)N]​e−jπ(f−f0​)(N−1)(4)

为什么是这个形式

公式(3)实际可以看做一个无限长的单频信号 x~1[n]\widetilde{x}_1[n]x1​[n] 与有限长矩形窗 w[n]w[n]w[n] 的乘积，定义 x~1[n]\widetilde{x}_1[n]x1​[n]为

x~1[n]=ej2πf0n,−∞⩽n⩽∞(6)\widetilde{x}_1[n] = e^{j2\pi f_0 n}, \quad -\infty \leqslant n \leqslant \infty \tag{6} x1​[n]=ej2πf0​n,−∞⩽n⩽∞(6)

则 x~1[n]\widetilde{x}_1[n]x1​[n] 的 DTFT 为

X~1(f)=∑l=−∞∞δ(f−f0−l)(7)\widetilde{X}_1(f) = \sum_{l = -\infty}^{\infty}\delta(f-f_0-l) \tag{7} X1​(f)=l=−∞∑∞​δ(f−f0​−l)(7)

定义 w[n]w[n]w[n]为

w[n]={1,0⩽n⩽N−10,otherwise(8) w[n] = \begin{cases} 1, & 0 \leqslant n \leqslant N-1 \\ 0, & \textrm{otherwise} \end{cases} \tag{8}w[n]={1,0,​0⩽n⩽N−1otherwise​(8)

则 w[n]w[n]w[n] 的 DTFT 为

W(f)=sin⁡[πfN]sin⁡[πf]e−jπf(N−1)(9)W(f) = \frac{\sin[\pi fN]}{\sin[\pi f]}e^{-j\pi f(N-1)} \tag{9} W(f)=sin[πf]sin[πfN]​e−jπf(N−1)(9)

当 N=10N = 10N=10 时，其归一化后的图像为这个函数类似 sinc 函数，称为 aliased sinc 函数，其瑞利宽度（峰值到第一零点）为 1/N1/N1/N, 其与 sinc 函数最大的区别是 aliased sinc 函数为周期函数。

因为有限长的单频信号 x1[n]x_1[n]x1​[n] 可以表示为

x1[n]=x~1[n]w[n](9)x_1[n] = \widetilde{x}_1[n]w[n] \tag{9} x1​[n]=x1​[n]w[n](9)

根据 DTFT 的相乘性质，X1(f)X_1(f)X1​(f) 即为 X~1(f)\widetilde{X}_1(f)X1​(f) 和 W(f)W(f)W(f) 的周期卷积，即

X1(f)=∫1X~1(u)W(f−u)du=∑l=−∞∞∫1δ(u−f0−l)W(f−u)du(10)\begin{aligned} X_1(f) &= \int_{1}\widetilde{X}_1(u)W(f-u) \mathrm{d}u\\ &= \sum_{l = -\infty}^{\infty}\int_{1}\delta(u-f_0-l)W(f-u)\mathrm{d}u \end{aligned} \tag{10}X1​(f)​=∫1​X1​(u)W(f−u)du=l=−∞∑∞​∫1​δ(u−f0​−l)W(f−u)du​(10)

式(10)中的积分可以在任意长度为 1 的区间内进行，最终可得

X1(f)=∑l=−∞∞W(f−f0−l)(11)X_1(f) = \sum_{l = -\infty}^{\infty}W(f-f_0-l) \tag{11} X1​(f)=l=−∞∑∞​W(f−f0​−l)(11)

x~1[n]\widetilde{x}_1[n]x1​[n] 和 w[n]w[n]w[n] 在主周期区间内的频谱分别为冲激函数和 aliased sinc 函数，由时域乘积对应频域卷积的性质，可知有限长单频信号 x1[n]x_1[n]x1​[n] 的频谱为冲激函数与 aliased sinc 函数的卷积，最终的结果表现为对 aliased sinc 函数的平移。而aliased sinc 函数的瑞利宽度为 1/N1/N1/N, NNN 越大，主瓣越窄；当 N→∞N \to \inftyN→∞ 时，矩形窗可对应连续情况下的直流信号，由傅里叶变换对

1↔δ(f)(12)1 \leftrightarrow \delta(f) \tag{12} 1↔δ(f)(12)

不难推知，极限情况下，有公式(5)的成立。

进一步的理解（匹配滤波）

实际中的信号 x[n]x[n]x[n] 如式(1)由多个单频信号叠加而成，理想情况下，当信号长度无限长时，归一化 DTFT X(f)X(f)X(f) 只在有频率定义的 fkf_kfk​ 处不为零，且幅值为信号的振幅 aka_kak​，故 X(f)X(f)X(f) 实为在频点 fff 处对信号 x[n]x[n]x[n] 中频率也为 fff 单频信号的匹配滤波。

匹配指的是：x[n]x[n]x[n] 由多个未知频率的信号叠加而成，我们用频率为 fff 的单频信号对 x[n]x[n]x[n] 做式(2)的 DTFT 运算。如果 x[n]x[n]x[n] 中也有频率为 fff 的单频信号分量，则运算的结果 X(f)X(f)X(f) 不为零，且幅值为原单频信号分量的振幅，这种情况称为匹配；否则不匹配情况下，运算结果为零。

滤波指的是：式(2)也可看成为将信号 x[n]x[n]x[n] 输入一个只能通过特定频点 fff 的滤波器 HfH_fHf​，输出的结果为 X(f)X(f)X(f).

匹配滤波指的是：我们不断调节滤波器 HfH_fHf​ 的频率，对信号 x[n]x[n]x[n] 做如式(2)的滤波。如果在某个特定频点 f0f_0f0​ 下，x[n]x[n]x[n] 有频率为 f0f_0f0​ 的单频信号分量，则滤波器 Hf0H_{f_0}Hf0​​ 的匹配输出结果不为零，且幅值为原单频信号分量的振幅；否则为零。

设由两个单频信号叠加而成的信号

x2[n]=0.8ej2π0n+0.5ej2π0.1n,0⩽n⩽N−1(13)x_2[n] = 0.8e^{j2\pi 0 n} + 0.5e^{j2\pi 0.1 n}, \quad 0 \leqslant n \leqslant N-1 \tag{13} x2​[n]=0.8ej2π0n+0.5ej2π0.1n,0⩽n⩽N−1(13)

对不同点数 NNN 的 x2[n]x_2[n]x2​[n] 做 DTFT, 其结果为 当信号点数 NNN 很小时，两个有限长单频信号 DTFT 的主瓣宽度都很宽，所以出现混叠。随着 NNN 的增大，主瓣越来越窄，最终越来越容易分开两个信号。

任意信号的 DFT

NNN 点信号的 KKK 点 DFT 为

X[k]=∑n=0N−1x[n]e−j2πk/Kn,k∈[0,K−1](14)X[k] = \sum_{n = 0 }^{N-1 }x[n]e^{-j2\pi k/K n}, \quad k \in [0, K-1] \tag{14} X[k]=n=0∑N−1​x[n]e−j2πk/Kn,k∈[0,K−1](14)

令

x′[n]={x[n],0⩽n⩽N−10,otherwise(15) x'[n] = \begin{cases} x[n], & 0 \leqslant n \leqslant N-1 \\ 0, & \textrm{otherwise} \end{cases} \tag{15}x′[n]={x[n],0,​0⩽n⩽N−1otherwise​(15)

则式(13)可变为

X[k]=∑n=−∞∞x′[n]e−j2πk/Kn,k∈[0,K−1](16)X[k] = \sum_{n = -\infty }^{\infty}x'[n]e^{-j2\pi k/K n}, \quad k \in [0, K-1] \tag{16} X[k]=n=−∞∑∞​x′[n]e−j2πk/Kn,k∈[0,K−1](16)

对比式(16)与式(2), 可知 DFT 实为有限长信号在频率为 fk=k/Kf_k = k/Kfk​=k/K 处的 DTFT. 具体来讲，NNN 个采样点序列 x[n]x[n]x[n] 的 DFT X[k]X[k]X[k] 为其 DTFT X(f)X(f)X(f) 的等间隔采样序列，其中 KKK 个频率采样在区间 f∈[0,1)f \in [0,1)f∈[0,1) 上是均匀分布的，即

X[k]=X(f)∣f=k/K,k∈[0,K−1](17)X[k] = X(f)|_{f = k/K}, \quad k \in [0, K-1] \tag{17} X[k]=X(f)∣f=k/K​,k∈[0,K−1](17)

所以 KKK 点 DFT 实际上为让信号 x[n]x[n]x[n] 通过 KKK 个频率为 fk=k/Kf_k = k/Kfk​=k/K 的滤波器组 HfkH_{f_k}Hfk​​, 输出结果为每个滤波器对应的 X[k], k∈[0,K−1]X[k], \, k \in [0, K-1]X[k],k∈[0,K−1].

结论

FFT 为 DFT 的快速算法，所以 FFT 是用匹配滤波的原理分析出信号频谱的。
