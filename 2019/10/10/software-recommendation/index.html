什么是电脑和软件？

推荐电脑要安装什么软件，首先得明白什么是电脑和软件。根据英文维基百科，电脑的定义：

A computer is a machine that can be instructed to carry out sequences of arithmetic or logical operations automatically via computer programming.

翻译过来就是：
计算机（俗称电脑）是指可以通过计算机编程（computer programming, 也叫计算机程序设计）自动执行算术或逻辑运算序列的机器。

软件的定义：

Computer software, or simply software, is a collection of data or computer instructions that tell the computer how to work.

计算机软件，或简称软件，是告诉计算机如何工作的数据或计算机指令的集合。

简单说来，电脑就是通过软件对数据进行运算（或计算）的机器。

因为人们有需求，比如要看图片、听歌、放视频。这些图片、音频、视频文件就是计算机要操作的数据；在屏幕上显示图片、对音视频进行解码以播放出来，就是计算机要进行的运算。程序员（programmer）通过编程（programming）开发出了软件（software），这些软件指导计算机进行这些运算以满足人们的需求。
